## Working with `uv`

If you are having issues with `uv`, install it using the following command:

```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

Then, you can use it by calling it with its full path:

```bash
$HOME/.local/bin/uv <command>
```
